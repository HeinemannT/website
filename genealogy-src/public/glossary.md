# Genealogy Glossary

## Key Terms

**Zupai (族派)**
The "Generation Poem" or ranking scheme. A poem where each character corresponds to a specific generation. It is used to select the middle character of a descendant's name (or the final character, depending on custom) to indicate their generational rank within the lineage.

**Tai-tou (抬頭)**
"Raising the head". An honorific formatting convention where the names of emperors, deities, or ancestors are started on a new line and elevated above the rest of the text. This visual break signifies deep respect.

**Jinshi (進士)**
"Advanced Scholar". The highest and most prestigious degree in the imperial examination system of Imperial China. Achieving this status usually led to high-ranking government appointments.

**Taboo Name (諱)**
A personal name used for ancestors that is considered too sacred to be spoken or written directly in common contexts. In records, it is introduced with the character "Hui" (諱) before the name to show reverence.

**Style Name (字 - Zi)**
A courtesy name bestowed upon adulthood. It is used by peers and friends, whereas the birth name is used by elders.

**Literary Name (號 - Hao)**
A pseudonym or art name chosen by the individual. It often reflects personal aspirations, artistic interests, or philosophical outlooks.

**Marginalia**
Notes, glosses, or comments written in the margins of a text. In this genealogy, they often denote page numbers or branch divisions (e.g., using characters from the Thousand Character Classic like "Tian", "Di", "Xuan", "Huang").

**Sanjie Temple (三界廟)**
"Temple of the Three Realms". A prominent local temple in the village where the clan would gather for worship and festivals.
